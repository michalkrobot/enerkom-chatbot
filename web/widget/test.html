Test stránka — vložený widget

Tato stránka načítá sestavený dist/widget.js (žádné CDN). Klikněte na tlačítko vpravo dole.

Spuštění lokálně: python -m http.server 8123 v adresáři web/widget, pak otevřete http://localhost:8123/test.html.

Pro reálné napojení nastavte data-api-url na běžící /api/chat. Bez backendu se po odeslání zobrazí přívětivá chybová hláška (to je očekávané).
